My Projects

Project: Robotic Car
Course: Automachines with Robotics
Duration: January 21th to Febuary 5th
Objective: Make a car that can travel 20 ft in the least amount of time
Type of Project: Group Project

Description: Our goal by the end of this project was to make a robot which can travel 20 ft distance in the least amount of time possible when a pushbutton switch is pressed to start it. When we were done we had the option to race multiple robots built by other teams against our robot (to determine which one was the fastest) or play against the clock and keep track of our time to see which teams robot reached the goal faster. My job in this project was being the Computer Engineer, while my teamates were doing their job me and my buddy were writing code which will move the robot foward as quick as possible. My grade on this project was a A+, becuase we beat another group of people in a race and all our code was perfect. Throughout this project I learned how to use VEX Components, and how to make a object travel faster with the help of a pushbutton and ROBOTC for VEX Components
